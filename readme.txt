Replace item id by key

== Description ==

This plugin is mainly for developper, It replaces post ID by key value.
You can easily move specific content, you don't need to change id in your code.
See screenshot samples


== Installation ==

1. Upload folder wp-manage-order to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin from settings menu
